# Cybersecurity ChatBot

A WPF C# cybersecurity chatbot application with two projects:

## Projects
- **Net_Bot_App** - Purple themed cybersecurity chatbot
- **CyberGuard Assistant** - Blue themed cybersecurity chatbot

## Features
- Sliding side menu with saved chats
- Keyword recognition and random responses
- Sentiment detection
- Memory and recall
- Animated typing indicator
- Chat saving and loading

## How to Run
1. Open the solution in Visual Studio
2. Build and run the project
3. Enter your name to start chatting
